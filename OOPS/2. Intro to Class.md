## What is a constructor?
- A constructor is a function that helps us in initializing a object with some default values

<mark style="background: #ADCCFFA6;">Some Properties</mark>
- The **name** is same as the ***class*** name
- The is no **return** type 
- Every class comes with a default constructor
## Constructor

```c++ fold title:'Constructors'
class Medhansh{
	public:
		Medhansh(){
			//constructor 
		}
};


```
### Types of constructors
- Default constructor
	```C++ fold title:'Default constructor'
A() {  
	x = 0;  
}
	```

- Parameterized constructor
	```C++ fold title:'Parameterized constructor'
A(int a) {  
	x = a;  
}
	```

- Copy constructor
	```C++ fold title:'Copy constructor'
A(const A& obj) {  
	x = obj.x;  
}
	```
	- **copying** the attribute of one object to another
	
- Move constructor (C++11+)
	```C++ fold title:'Move constructor'
A(A&& obj) {  
	x = obj.x;  
	obj.x = 0;  
}

&& -> rvalue reference
& → lvalue reference

	```
  
## `This`  keyword

## Names 

In the above program thing like `name,age` these are the **attributes** while `sayMyName` is a **method** , **behaviors** of the object 

## Destructor 

- to deallocate the memory  used by the object 

```C++ fold title:'Destructor'
class students{
	~student(){
		//deallocate the memory object was using
	}
};
```

<mark style="background: #ADCCFFA6;">Some Properties</mark>
- The **name** is same as the ***class*** name
- The is no **return** type 
- Every class comes with a default destructor
