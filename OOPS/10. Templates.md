# What
- Templates in C++ are a **compile-time mechanism** that lets you write **generic** **code** once and use it with many different types, allowing the compiler to generate type-specific versions automatically.
- we write code without writing code
- **compile-time code generators**.

```cpp
//for int 
int min(int b,int a){
    return ( b>a ) ? a : b ;
}

//for double
double min(double b,double a){
    return ( b>a ) ? a : b ;
}

//for float
float min(float b,float a){
    return ( b>a ) ? a : b ;
}
```

```cpp
template<typename T>
T min(T b,T a){
    return ( b>a ) ? a : b ;
}
cout<<min(2,3); // 2
cout<<min(2.1,3); //2.1
cout<<min(2.45,3.333);//3.333
cout<<min("Medhansh","Someone"); //Medhansh
cout<<min('a','b'); // a
```

### Two ways to call a template
### Explicit type

```c++
max<int>(10, 20);
```

You tell the compiler the type.

### Type deduction

```c++
max(10, 20);
```

The compiler figures out `T` automatically.

This is called **template argument deduction**.
