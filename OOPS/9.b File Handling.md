## 🧠 LEVEL 1 — Foundations (but not trivial)

### 1. Log Analyzer
**Input:**  
A log file with lines like:
```
2026-04-06 ERROR Disk full
```

**Tasks:**
- Count occurrences of each log level (`INFO`, `ERROR`, `WARNING`)
- Output summary to another file

> [!tip]
> Focus: parsing + reading line by line

---

### 2. CSV Student Processor
**File Format:**
```
name,marks
```

**Tasks:**
- Compute average
- Find topper
- Create a new file with grades (A/B/C)

> [!tip]
> Focus: string splitting + structured reading

---

### 3. Word Frequency Counter (Case-insensitive)
**Input:**  
Large text file

**Tasks:**
- Count frequency of each word
- Ignore punctuation
- Output sorted result

> [!tip]
> Focus: maps + cleaning input

---

## ⚙️ LEVEL 2 — Intermediate (real thinking starts)

### 4. File Merge Tool
**Input:**  
Two sorted files

**Task:**
- Merge into a third sorted file

> [!tip]

### 5. Persistent TODO Manager
**Commands:**
```
add Buy milk
list
done 2
```

**Task:**
- Store tasks in file permanently

> [!tip]
> Focus:

### 6. Binary File Student Database
**Struct:**
```cpp
struct Student {
    int id;
    char name[50];
    float marks;
};
```

**Tasks:**
- Add record
- Search by ID
- Update marks

> [!important]
> Binary file I/O (`fstream`, `seekg`, `seekp`)
