# Study Things 📚

A collection of notes, PDFs, reference materials, cheat sheets, and study resources that I found myself sharing repeatedly with friends, classmates, and anyone starting their learning journey.

Instead of digging through chats, drives, and old messages every time someone asks for a resource, everything is organized here in one place.

## What's Inside?

* 📄 Notes
* 📚 PDFs
* 📝 Cheat Sheets
* 💻 Programming Resources
* 🔗 Useful Links
* 🎓 Learning Material I've Collected Along the Way

## Why This Repository Exists

Over time, I realized I was sending the same files and recommendations to different people again and again.

This repository solves that problem by acting as a centralized resource hub:

* Easier to share
* Easier to maintain
* Easier to update
* Easier for others to discover useful material

## Who Is This For?

Anyone who:

* Wants study resources quickly
* Is learning Computer Science
* Needs reference material for coursework
* Prefers having everything in one organized place

## Contributing

Found a mistake, outdated file, or better resource?

Feel free to open an issue or submit a pull request.
